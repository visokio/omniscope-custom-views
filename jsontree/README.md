# ![](icon.svg) JSON Tree

Custom view that renders JSON content as a navigable, expandable tree structure using the [jsTree](https://www.jstree.com/) library.

## Use case

Visualise raw JSON stored in a dataset field as a collapsible tree.

Useful when your dataset contains one or more records with JSON strings and you want to inspect the structure, drill into nested objects, or quickly understand the shape of the data.

## Settings

- **JSON Field**: The field containing the raw JSON data to be parsed and displayed as a tree.

## Libraries

- [jsTree](https://www.jstree.com/)
